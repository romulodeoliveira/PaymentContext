# Introdução

Este projeto foi criado durante meus estudos sobre DDD e arquitetura de software. O objetivo é aplicar os conceitos de OOP, DDD e CQRS para modelar um contexto de pagamentos.

# Tecnologias utilizadas

- .NET 8
- Asp.NET Core
- Flunt
